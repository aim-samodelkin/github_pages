# ROI Dashboard — ТМХ × ИИ-агенты

Интерактивный дашборд для визуализации ROI от внедрения ИИ-агентов.

**Живая страница:** https://aim-samodelkin.github.io/github_pages/

---

## Как обновить страницу

После любых изменений в `roi_dashboard.html` выполнить в терминале:

```bash
cp roi_dashboard.html index.html
git add index.html
git commit -m "Update dashboard"
git push
```

Через **1–2 минуты** изменения появятся на сайте.

---

## Структура

| Файл | Описание |
|------|----------|
| `roi_dashboard.html` | Исходник дашборда (редактировать его) |
| `index.html` | Копия для GitHub Pages (генерируется из исходника) |
| `ROI_анализ.md` | Методология и расчёты |
| `расчет_трудозатрат.csv` | Трудозатраты без ИИ |
| `расчет_трудозатрат_ИИ.csv` | Трудозатраты с ИИ |
| `шаги реализации.md` | План внедрения |
